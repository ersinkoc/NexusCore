# @nexuscore/cli

Interactive CLI tool for scaffolding NexusCore projects.

## Overview

The NexusCore CLI provides an interactive way to create new projects based on the NexusCore boilerplate.

## Usage

```bash
npx create-nexuscore my-app
```

Or with interactive prompts:

```bash
npx create-nexuscore
```

## Features

- Interactive project setup
- Choice of package manager (pnpm, npm, yarn)
- Optional Docker configuration
- Optional demo e-commerce module
- Automatic dependency installation
- Environment file generation
