# @nexuscore/ui

Shared UI component library for NexusCore applications.

## Overview

This package contains reusable React components based on Shadcn/UI design principles with TailwindCSS.

## Usage

```tsx
import { Button } from '@nexuscore/ui';

function MyComponent() {
  return (
    <Button variant="primary" size="md">
      Click me
    </Button>
  );
}
```

## Components

- **Button**: A versatile button component with multiple variants and sizes

More components will be added in Phase 4 of development.
